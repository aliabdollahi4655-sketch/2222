فرمول‌های فیزیک دبیرستان

مجموعه کامل فرمول‌های فیزیک پایه دهم، یازدهم و دوازدهم

🚗

سرعت متوسط

$v = \frac{s}{t}$

v: سرعت، s: مسافت، t: زمان

📈

شتاب

$a = \frac{v - v_0}{t}$

a: شتاب، v: سرعت نهایی، v₀: سرعت اولیه

📏

معادله حرکت

$s = v_0 t + \frac{1}{2}at^2$

s: جابجایی، v₀: سرعت اولیه، a: شتاب

⚡

قانون دوم نیوتن

$F = ma$

F: نیرو، m: جرم، a: شتاب

🔄

انرژی جنبشی

$K = \frac{1}{2}mv^2$

K: انرژی جنبشی، m: جرم، v: سرعت

⬆️

انرژی پتانسیل

$U = mgh$

U: انرژی پتانسیل، m: جرم، g: شتاب گرانش، h: ارتفاع

🌊

سرعت موج

$v = f\lambda$

v: سرعت موج، f: فرکانس، λ: طول موج

🔄

دوره نوسان

$T = \frac{1}{f}$

T: دوره، f: فرکانس

🎯

آونگ ساده

$T = 2\pi\sqrt{\frac{l}{g}}$

T: دوره، l: طول آونگ، g: شتاب گرانش

🌡️

تبدیل دما

$C = \frac{5}{9}(F - 32)$

C: سلسیوس، F: فارنهایت

🔥

گرمای ویژه

$Q = mc\Delta T$

Q: گرما، m: جرم، c: گرمای ویژه، ΔT: تغییر دما

💨

قانون گاز ایده‌آل

$PV = nRT$

P: فشار، V: حجم، n: تعداد مول، R: ثابت گاز، T: دما

⚡

قانون اهم

$V = IR$

V: ولتاژ، I: جریان، R: مقاومت

🔋

توان الکتریکی

$P = VI = I^2R = \frac{V^2}{R}$

P: توان، V: ولتاژ، I: جریان، R: مقاومت

🧲

نیروی مغناطیسی

$F = BIL\sin\theta$

F: نیرو، B: میدان مغناطیسی، I: جریان، L: طول سیم

🔍

قانون عدسی

$\frac{1}{f} = \frac{1}{d_o} + \frac{1}{d_i}$

f: فاصله کانونی، d₀: فاصله جسم، dᵢ: فاصله تصویر

🌈

قانون اسنل

$n_1\sin\theta_1 = n_2\sin\theta_2$

n: ضریب شکست، θ: زاویه تابش/انکسار

📏

بزرگنمایی

$m = \frac{h_i}{h_o} = -\frac{d_i}{d_o}$

m: بزرگنمایی، h: ارتفاع، d: فاصله

⚛️

انرژی فوتون

$E = hf = \frac{hc}{\lambda}$

E: انرژی، h: ثابت پلانک، f: فرکانس، c: سرعت نور

🚀

معادله انیشتین

$E = mc^2$

E: انرژی، m: جرم، c: سرعت نور

🌊

طول موج دوبروی

$\lambda = \frac{h}{p} = \frac{h}{mv}$

λ: طول موج، h: ثابت پلانک، p: تکانه، m: جرم، v: سرعت
